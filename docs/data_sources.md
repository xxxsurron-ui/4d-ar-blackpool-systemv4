# Data Sources

## Core Datasets
- Blackpool coastline
- DEM / LiDAR
- Transport networks
- Buildings and infrastructure
- Environmental data (tides, weather, air quality)

## Temporal Data
- Time-series for tides, traffic, weather, environmental indicators
- Stored in `data/temporal/`

## Notes
- Raw data → `data/raw/`
- Processed data → `data/processed/`
- Blackpool curated layers → `data/blackpool/`
